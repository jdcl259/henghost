## 有什么要注意的坑？

用了这么久，也遇到过一些小问题，提前说说让大家少踩坑。

**IP更换政策**

1核1G和1核2G的配置，买了之后不支持免费更换IP（除非IP本身有问题）。如果你非要换IP，得额外付费。其他配置可以免费换一次，再换就得花钱了。

所以如果你是做站群或者对IP有特殊要求的，建议买之前先问清楚IP段的情况。

**不能换机房**

买了香港的就只能用香港的，想换到日本或者美国是不行的。所以下单之前一定要想清楚自己的业务主要面向哪个地区的用户。

**特价款不退款**

活动期间的特价商品基本不支持退款。如果你真有特殊情况要退，会按原价折算实际使用时长，然后退剩下的钱。说白了就是把优惠部分给抹掉了，所以买之前最好确认清楚配置和需求。

**Windows系统的坑**

如果你要装Windows Server，1核1G的配置肯定不够用。Windows系统本身就占内存，至少得2核4G起步。而且Windows授权费比Linux贵，套餐价格会略高一些。

**备案问题**

虽然海外主机不用备案，但如果你的业务主要面向国内用户，后期可能还是得考虑备案。有些支付接口、广告联盟会要求网站必须备案才能接入。这个要提前规划好。
